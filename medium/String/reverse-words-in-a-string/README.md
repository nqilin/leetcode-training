# [LeetCode 151 - Reverse Words in a String](https://leetcode.com/problems/reverse-words-in-a-string/)
> Difficulty: Medium | Tags: String

## 🎯 Problem Description
Given an input string `s`, reverse the order of the **words**.
A word is defined as a sequence of non-space characters.
Return the reversed string with **single space between words and no leading/trailing spaces**.

## Examples
- Input: "the sky is blue"
- Output: "blue is sky the"

## 💡 Solution Ideas
1. Split the string into words (automatically skip multiple spaces).
2. Reverse the list of words.
3. Join with single spaces.
